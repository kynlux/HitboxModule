Hitbox

Hitbox main class that will be used for your hitboxes and their manipulations

Hitbox.new()

Hitbox.new() accepts parameters

  • cframe (CFrame) - CFrame of Hitbox
  • size (Vector3) - Hitbox Size
  • overlap (OverlapParams) (not neccesary) - OverlapParams for Hitbox
  • owner (Instance) (not neccesary) - Owner of Hitbox (the owner of the hitbox can't be detected by hitbox)
  • visual (boolean) (not neccesary) - Show visual of hitbox

Hitbox.new() is used to initialize a hitbox

Hitbox:GetHits()

Hitbox:GetHits() accepts parameters

  • sorted (boolean) (not neccesary) - Make returning table much useful and clear
  • clean (boolean) (not neccesary) - Clean self.Hits after getting executing method

Hitbox:GetHits() is used to get self.Hits

Hitbox:UpdateTargets()

Hitbox:UpdateTargets() accepts parameters

  • delayTime (number) (not neccesary) - Delay time before clearing each target after it is detected
  • clean (Vector3) (not neccesary) - Clean self.Hits before getting targets

Hitbox:UpdateTargets() is used for a single update of self.Hits (i.e. updating the table with targets)

Hitbox:SmoothMove()

Hitbox:SmoothMove() accepts parameters

  • cframe (CFrame) - New cframe for hitbox
  • moveTime (number) - Time to move hitbox
  • targets (boolean) (not neccesary) - Update targets or not
  • delayTime (number) (not neccesary) - Delay time before clearing each target after it is detected
  • clean (Vector3) (not neccesary) - Clean self.Hits before getting targets

Hitbox:SmoothMove() is used to move the hitbox around

Hitbox:Clean()

Hitbox:Clean() accepts parameters

  • delayTime (number) (not neccesary) - Delay before cleaning

Hitbox:Clean() is used to clean self.Hits
